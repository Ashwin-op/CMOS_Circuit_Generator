# CMOS Circuit Generator

Converting Boolean Expressions to CMOS Circuits.

## Requirements

- [ttable](https://tt.brianwel.ch/en/latest/)

## Usage

```shell
pip3 install ttable

python3 main.py
```

## Supported Operators

Available operators: and, iff, impl, nand, nor, not, nxor, or, xnor, xor


## Example Input

Example: A and B or not C xor D
